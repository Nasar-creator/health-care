# Help Center API Assignment

## Instructions

1. **Clone the Repository:**
   ```bash
   gh repo clone Nasar-creator/health-care
   cd health-care
   ```

2. **Run:**
   - To run both in same Time run the cmd ``` npm run dev ``` in the root folder.
    - **OR**
    ```bash
      npm run server
      npm run client
     ```
